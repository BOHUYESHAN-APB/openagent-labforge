## 关键改进

### 安装流程
**之前：**
```bash
# 不明确的流程
bun run build
bun pm pack
# 然后...？使用 file:/// 路径？
```

**现在：**
```bash
# 清晰的步骤
git clone https://github.com/BOHUYESHAN-APB/openagent-labforge.git
cd openagent-labforge
bun install
bun run build
npm pack
npm install -g ./bohuyeshan-openagent-labforge-core-<version>.tgz

# 配置（使用包名，不是 file:// 路径）
{
  "plugin": [
    "@bohuyeshan/openagent-labforge-core@<version>",
    "opencode-pty@0.3.2"
  ]
}
```

### 配置方法
**之前：**
- 推荐使用 `file:///ABSOLUTE/PATH/TO/openagent-labforge`
- 问题：插件可能无法正确加载

**现在：**
- 推荐使用 `@bohuyeshan/openagent-labforge-core@<version>`
- 明确警告不要使用 `file:///` 路径
- 提供完整的验证步骤
