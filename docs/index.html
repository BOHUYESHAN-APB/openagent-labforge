Quick Start

Terminal
$# Clone repository
$git clone https://github.com/BOHUYESHAN-APB/openagent-labforge-bio.git
✓ Repository cloned
$# Build
$cd openagent-labforge-bio && bun install && bun run build
✓ Build complete
$# Add to ~/.config/opencode/opencode.json
$"plugin": ["file:///path/to/openagent-labforge-bio"]
$# Launch
$opencode
🤖 ExtendAI Lab is ready!
