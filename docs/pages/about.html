About ExtendAI Lab

AI-Powered Bioinformatics Agent Orchestration

🎯 Mission

ExtendAI Lab aims to bridge the gap between AI coding assistants and domain-specific bioinformatics expertise. We provide a plugin system that brings 617 specialized skills to OpenCode, enabling researchers to leverage AI for complex bioinformatics workflows.

🏗️ Architecture

Built as an OpenCode plugin, ExtendAI Lab uses a multi-agent architecture with 15 specialized agents. The system features persistent memory, auto-continue functionality, and seamless integration with bioinformatics tools.

📚 Skills

617 bioinformatics skills across 88 categories, covering genomics, transcriptomics, proteomics, single-cell analysis, clinical bioinformatics, and more. Each skill provides detailed workflows and best practices.

📝 Academic Writing

Complete academic writing pipeline with GB/T 7714-2015 citation support, MD→DOCX conversion, proper font handling (SimSun/TNR/SimHei), and REF field code cross-references.
