关于 ExtendAI Lab

AI驱动的生物信息学智能体编排

🎯 使命

弥合 AI 编程助手与领域特定生物信息学专业知识之间的差距。提供一个插件系统，将617个专业技能带到 OpenCode，使研究人员能够利用 AI 处理复杂的生物信息学工作流。

🏗 架构

作为 OpenCode 插件构建，ExtendAI Lab 使用包含15个专业智能体的多智能体架构。具有持久化记忆、自动继续功能，并与生物信息学工具无缝集成。

📚 技能

覆盖88个分类的617个生物信息学技能，涵盖基因组学、转录组学、蛋白质组学、单细胞分析、临床生物信息学等。

📝 学术写作

完整的学术写作管线，支持 GB/T 7714-2015 引用格式、MD→DOCX 转换、正确的字体处理（宋体/TNR/黑体）以及 REF 字段代码交叉引用。

设计

设计理念

主智能体优先

主智能体保留完整的上下文和控制权。仅在必要时生成子会话，结果反馈回主会话以保持连贯性。

成本优化

尽可能使用 load_agent_instructions 代替生成子会话。保持缓存并减少 Token 使用。

跨窗口状态

计划文件、boulder.json 和会话 ID 跨窗口持久化。钩子注入的上下文确保隔离上下文中的状态恢复。

独立 MCP

每个窗口获得独立的 MCP 服务器实例。跨进程无共享 stdio 传输冲突。
